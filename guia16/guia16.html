Laboratorio 16: Creación de un Formulario de Registro Completo con Imágenes

En este laboratorio, aprenderás a crear un formulario de registro completo que permite al usuario ingresar sus datos personales y subir una imagen de perfil. También aplicaremos validaciones de datos con JavaScript y mejoraremos la apariencia visual usando CSS.

Punto 1: Estructura Básica del Formulario en HTML

Comienza creando un archivo llamado laboratorio16.html y añade el siguiente código para la estructura básica del formulario y el menú de navegación:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="styles.css"> 
    <title>Formulario de Registro</title>
</head>
<body>

    <!-- Menú de Navegación -->
    <header style="background-color: #333; padding: 10px 0;">
        <nav>
            <ul style="list-style: none; display: flex; justify-content: center; margin: 0; padding: 0;">
                <li style="margin: 0 15px;"><a href="#" style="color: white; text-decoration: none; font-weight: bold;">Inicio</a></li>
                <li style="margin: 0 15px;"><a href="#" style="color: white; text-decoration: none; font-weight: bold;">Acerca de</a></li>
                <li style="margin: 0 15px;"><a href="#" style="color: white; text-decoration: none; font-weight: bold;">Servicios</a></li>
                <li style="margin: 0 15px;"><a href="#" style="color: white; text-decoration: none; font-weight: bold;">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <form class="formulario" action="#" method="POST" enctype="multipart/form-data" id="registroForm">
        <h2>Registro de Usuario</h2>
        <label>Nombre:</label>
        <input type="text" name="nombre" required>

        <label>Correo Electrónico:</label>
        <input type="email" name="email" required>

        <label>Contraseña:</label>
        <input type="password" name="password" required>

        <label>Imagen de Perfil:</label>
        <input type="file" name="imagen" id="imagen" accept="image/*" onchange="mostrarImagen(event)" required>

        <img id="ver-imagen" src="#" alt="Vista previa de la imagen" />

        <button type="submit">Registrar</button>
    </form>
</body>
</html>

Punto 2: Aplicar Estilos CSS

Crea un archivo llamado styles.css y añade el siguiente código para personalizar el formulario y hacer que se vea visualmente atractivo:

Punto 3: Validación de Formulario con JavaScript

Crea un archivo llamado script.js y añade el siguiente código para implementar la validación de datos, mostrar la vista previa de la imagen y la alerta de registro exitoso:


            // script.js

            // Función para mostrar la vista previa de la imagen seleccionada
            function mostrarImagen(event) {
                const imagen = document.getElementById('ver-imagen'); // Selecciona el elemento de imagen de vista previa
                imagen.src = URL.createObjectURL(event.target.files[0]); // Crea una URL temporal para mostrar la imagen seleccionada
                imagen.style.display = 'block'; // Muestra la imagen en la página
            }
            
            // Función para validar el formulario antes de enviarlo
            document.getElementById('registroForm').addEventListener('submit', function(event) {
                // Obtiene los valores de cada campo del formulario
                const nombre = document.querySelector('input[name="nombre"]').value;
                const email = document.querySelector('input[name="email"]').value;
                const password = document.querySelector('input[name="password"]').value;
            
                // Verifica que todos los campos estén completos
                if (nombre === '' || email === '' || password === '') {
                    alert('Por favor completa todos los campos.'); // Muestra una alerta si algún campo está vacío
                    event.preventDefault(); // Evita el envío del formulario
                } else {
                    alert('Registro exitoso'); // Muestra una alerta de éxito si todos los campos están completos
                }
            });

No olvides enlazar script.js al final de laboratorio16.html antes de </body>:

<script src="script.js"></script>

Punto 4: Pruebas del Formulario

Abre el archivo laboratorio16.html en un navegador web y prueba el formulario. Intenta subir una imagen de perfil, llena los campos y envía el formulario para asegurarte de que todo funcione correctamente.
